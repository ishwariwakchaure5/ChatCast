# ChatCast

ChatCast is a Python-based chat application with a client interface.
It supports real-time communication using a structured backend, testing framework, and documentation.

## Features

- Real-time chat server
- Client-side interface
- Modular backend design
- Automated test cases
- Clean and organized project structure

## Project Structure

chatcast-client/   – client side code  
docs/              – project documentation  
tests/             – test files  
server.py          – main server file  
requirements.txt   – project dependencies  

## Technologies Used

- Python
- PyTest
- Git & GitHub

## How to Run the Project

1. Install dependencies:
pip install -r requirements.txt

2. Run the server:
python server.py

## Notes

- Virtual environment files are ignored using .gitignore
- Make sure Python is installed before running the project
